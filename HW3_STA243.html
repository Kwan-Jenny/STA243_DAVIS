Then the log-likelihood function is, \[ l_c(\alpha) = -2n \log \alpha - \frac{1}{\alpha^2} \sum_{i=1}^n y_i^2 + c \]

Define an expression for \(Q(\alpha,\alpha_{(k)})\), \[ Q(\alpha,\alpha_{(k)}) = \mathbb{E}_{\alpha_{(k)}} [l_c(\alpha)|\mathbf{X}] = -2n \log \alpha - \frac{1}{\alpha^2} \sum_{i=1}^n \mathbb{E}_{\alpha_{(k)}} [y_i^2|\mathbf{X}] \]

\[ \mathbb{E}_{\alpha_{(k)}} [y_i^2|\mathbf{X}] = \mathbb{E}_{\alpha_{(k)}} [y_i^2|y_i=y_i] = y_i^2 \]
